# Planova AI

Planova AI is a student productivity website that helps students organize their study schedules, manage learning goals, and improve productivity using AI tools.

This project was developed for the Selected Topics course at Almaarefa University.

---

# Project Idea

Many students struggle with:
- Managing their study time
- Organizing subjects
- Balancing study and personal time
- Staying productive

The purpose of Planova AI is to make studying more organized and easier for students through AI-based planning tools.

---

# Main Features

## Study Planner
The user can:
- Add subjects
- Choose difficulty levels
- Select busy days
- Generate a weekly study schedule

---

## Skill Planner
Users can:
- Add skills they want to learn
- Track progress
- Organize learning goals

---

## AI Assistant
The website includes an AI assistant connected to OpenAI API.

The AI can:
- Analyze schedules
- Give study recommendations
- Help improve productivity

---

## Dashboard
The dashboard shows:
- Productivity statistics
- Study progress
- Charts and analytics

# How To Run The Project

## Install dependencies

```bash
npm install
```

## Start frontend

```bash
npm run dev
```

## Start backend

```bash
npm run server
```

---

# Environment Variables

Create a `.env` file and add:

```env
OPENAI_API_KEY=your_api_key
PORT=3001
```
